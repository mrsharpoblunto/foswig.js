Generate awesome Javascript project names

Have you just put the finishing touches on a sweet open source JS library that is going to be the toast of Hacker News but can't think of a suitably cool name?

Don't worry, This site has aggregated a list of all the popular JavaScript library names on GitHub and put them into a Markov Chain which allows for the creation of convincing randomly generated names that (for the most part) sound like real words. The code to generate the Markov Chain from the input dictionary is called Foswig.js and is available here.
